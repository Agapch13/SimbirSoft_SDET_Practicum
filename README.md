ТЕСТ-КЕЙС №1: Проверка сортировки в категориях
=

## Предусловие:
Открыта главная страница магазина https://automationteststore.com/.
<br >В категории выбранного раздела присутствует не менее 4 товаров.

## Шаги выполнения:
- Перейти в категорию "Apparel & Accessories".
- Выбрать сортировку "Name A - Z" (по имени по возрастанию).
- Выбрать сортировку "Name Z - A" (по имени по убыванию).
- Выбрать сортировку "Price Low > High" (по цене по возрастанию).
- Выбрать сортировку "Price High > Low" (по цене по убыванию).
## Ожидаемый результат:
1. Товары отсортированы по названию от А до Я.
2. Товары отсортированы по названию от Я до А.
3. Товары отсортированы по цене от дешёвых к дорогим.
4. Товары отсортированы по цене от дорогих к дешёвым.

ТЕСТ-КЕЙС №2: Проверка сортировки в категориях
=

## Предусловие:
Открыта главная страница магазина https://automationteststore.com/.
<br> Перейти в корзину. В корзине не содержится ни одного товара, отображается сообщение "Your shopping cart is empty!"

## Шаги выполнения:
- В поле поиска ввести "shirt" и нажать Enter.
- Отсортировать результаты поиска по наименованию (Name A - Z).
- Запомнить 2-й и 3-й товар в выдаче. Для каждого задать случайное количество (от 1 до 5) и нажать "Add to Cart".
- Перейти в корзину (кликнуть на иконку корзины → "View Cart").
- Найти самый дешёвый товар в корзине (по цене за единицу).
- Удвоить его количество.
- Проверить итоговую сумму корзины.
## Ожидаемый результат:
1. Открыта страница результатов поиска, содержащая товары со словом "shirt" в названии.
2. Результаты отсортированы по имени по возрастанию.
3. Товары добавлены в корзину с указанным количеством.
4. Открыта страница корзины с добавленными товарами.
5. Самый дешёвый товар найден.
6. Количество товара обновлено, цена за позицию пересчитана.
7. Итоговая сумма корзины соответствует расчётной (сумма всех цена × количество).

ТЕСТ-КЕЙС №3: Проверка корзины (удаление чётных товаров)
=

## Предусловие:
Открыта главная страница магазина https://automationteststore.com/.
<br>Перейти в корзину. В корзине не содержится ни одного товара, отображается сообщение "Your shopping cart is empty!"

## Шаги выполнения:
- Выбрать 5 случайных различных товаров с главной страницы.
- Для каждого задать случайное количество (от 1 до 5) и добавить в корзину.
- Перейти в корзину.
- В таблице товаров удалить все товары, стоящие на чётных позициях (2-й и 4-й по порядку).
- Рассчитать ожидаемую итоговую стоимость (сумма оставшихся цена × количество).
- Сравнить с итоговой суммой на странице корзины.

## Ожидаемый результат:
1. В корзину добавлено 5 товаров с разным количеством.
2. Открыта страница корзины с 5 позициями.
3. Удалены 2-й и 4-й товары, в корзине осталось 3 товара.
4. Ожидаемая сумма рассчитана.
5. Итоговая сумма на странице совпадает с ожидаемой.
